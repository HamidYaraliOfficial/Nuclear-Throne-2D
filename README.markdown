# Nuclear Throne 2D

## 🇬🇧 English

### Overview
Nuclear Throne 2D is a fast-paced top-down shooter inspired by classic roguelike action games. Built with Python and PyQt6, the project features wave-based combat, multiple weapons, enemy varieties, boss encounters, experience and leveling systems, pickups, particle effects, dynamic themes, and multilingual support.

### Features
- Top-down 2D action gameplay
- Multiple weapons (Pistol, Shotgun, Minigun)
- Wave-based enemy system
- Boss battles
- XP and leveling progression
- Health, ammo, XP, and weapon pickups
- Dark and Light themes
- Multilingual interface (English, Persian, Chinese)
- Particle effects and visual feedback
- FPS counter and statistics panel
- Responsive PyQt6 desktop interface

### Requirements
- Python 3.10 or newer
- PyQt6

### Installation

#### Install Required Libraries
```bash
pip install PyQt6
```

#### Run the Project
```bash
python "Nuclear Throne 2D.py"
```

### Controls

| Key | Action |
|------|---------|
| W A S D | Move |
| Mouse | Aim |
| Left Click | Shoot |
| R | Reload |
| 1 / 2 / 3 | Switch Weapon |
| ESC | Pause / Resume |

### Gameplay
Survive endless enemy waves, defeat bosses, gain experience, unlock higher levels, improve survivability, and achieve the highest score possible.
